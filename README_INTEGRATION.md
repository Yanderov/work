# 🎮 WildClient + Yanderov Integration

## ✅ Интеграция завершена на 100%!

Этот проект содержит успешную интеграцию продвинутых функций из Yanderov клиента в WildClient.

---

## 📦 Что было интегрировано

### 1. KillAura (Модуль Aura)
Добавлено **6 новых режимов наведения** и **6 новых настроек** из Yanderov в существующий модуль Aura.

**Новые режимы:**
- FunTime - плавная ротация с рандомизацией
- Matrix - оптимизирован для Matrix античита
- SpookyTime - средняя скорость ротации
- HolyWorld - средняя скорость ротации
- Advanced - полностью настраиваемый режим
- Smooth - очень плавная ротация

**Новые настройки:**
- ClickMode (1.9 / 1.8)
- CPS (1-20 для режима 1.8)
- HitChance (1-100%)
- HorizontalSpeed (для Advanced)
- VerticalSpeed (для Advanced)
- Randomization (для Advanced)

### 2. Speed (Модуль Speed)
Добавлено **6 новых режимов** и **16 новых настроек** из Yanderov в существующий модуль Speed.

**Новые режимы:**
- Grim - ускорение от столкновений с сущностями
- HolyWorld - умное ускорение в зависимости от скорости цели
- FunTime One Block - оптимизирован для One Block режима
- Meta - продвинутый режим с множеством настроек
- FunSky HVH - режим для HvH серверов
- 45Degree - специальный режим с углом 45°

**Новые настройки:**
- Для Grim: AuraBoost, Strength
- Для Meta: Speed 0-3+, DamageBoost, BoostDuration, MinDamageSpeed, CollisionBoost, CollisionPower
- Для FunSky HVH: DamageBoost, BoostDuration, MinDamageSpeed
- Для HolyWorld: LowBoost, HighBoost, ScanRadius

## 🚀 Быстрый старт

### Использование Aura:

1. Откройте меню модулей в игре
2. Перейдите в категорию **Combat**
3. Найдите модуль **Aura**
4. Выберите режим в настройке **Mode**:
   - `FunTime` - для плавной ротации
   - `Matrix` - для Matrix античита
   - `Advanced` - для полной настройки
   - `Smooth` - для очень плавной ротации
5. Настройте **ClickMode** (1.8 или 1.9)
6. Настройте **CPS** и **HitChance**

### Использование Speed:

1. Откройте меню модулей в игре
2. Перейдите в категорию **Movement**
3. Найдите модуль **Speed**
4. Выберите режим в настройке **Mode**:
   - `Grim` - для Grim античита
   - `Meta` - продвинутый режим
   - `HolyWorld` - умное ускорение
   - `FunSky HVH` - для HvH серверов
5. Настройте параметры выбранного режима

## 🎯 Особенности интеграции

### Интеграция между модулями:
Модуль Speed интегрирован с модулем Aura:
- Режим Grim усиливается при наличии цели в Aura
- Режим Meta усиливается при наличии цели в Aura
- Автоматическое определение активности Aura

### Продвинутые системы:

**Damage Boost:**
- Ускорение при получении урона
- Настраиваемая длительность буста
- Настраиваемая минимальная скорость

**Collision Boost:**
- Ускорение от столкновений с сущностями
- Настраиваемая сила столкновений
- Интеграция с Aura для усиления

**HitChance:**
- Шанс удара в процентах
- Честная рандомизация
- Обход античитов

**Режим 1.8:**
- Настраиваемый CPS (1-20)
- Независимый от кулдауна Minecraft
- Таймер для точности
