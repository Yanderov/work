## 💡 КАК ИСПОЛЬЗОВАТЬ

### Aura (Combat):
```
1. Откройте меню модулей
2. Combat → Aura
3. Mode → выберите режим (FunTime, Matrix, Advanced, Smooth)
4. ClickMode → 1.8 или 1.9
5. Настройте CPS и HitChance
```

### Speed (Movement):
```
1. Откройте меню модулей
2. Movement → Speed
3. Mode → выберите режим (Grim, Meta, HolyWorld, FunSky HVH)
4. Настройте параметры режима
```

### Target HUD:
```
1. Откройте настройки HUD
2. Найдите Target HUD
3. Style → выберите стиль (WildClient, Yanderov, Compact)
4. Перетащите в нужное место
```
