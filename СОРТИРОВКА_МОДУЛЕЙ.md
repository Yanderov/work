## 📝 ПРАВИЛА ДОБАВЛЕНИЯ НОВЫХ МОДУЛЕЙ

При добавлении нового модуля:

1. **Определите категорию** модуля (COMBAT, MOVEMENT, RENDER, PLAYER, UTIL)
2. **Найдите секцию** этой категории в `ModuleManager.java`
3. **Вставьте модуль** в алфавитном порядке внутри категории
4. **Проверьте компиляцию** командой `./gradlew build`

### Пример:

Если добавляете модуль `NewModule` в категорию COMBAT:

```java
// COMBAT (по алфавиту)
new AntiBot(),
new AntiTarget(),
new Aura(),
// ... другие модули
new NewModule(),  // <-- вставить в алфавитном порядке
// ... остальные модули
```
