## 🎮 ФУНКЦИОНАЛ YANDEROV KILLAURA

### Режимы наведения (AimMode):
1. **FunTime** - плавная ротация с рандомизацией (для FunTime сервера)
2. **Matrix** - оптимизирован для Matrix античита
3. **Snap** - мгновенная ротация на цель
4. **SpookyTime** - средняя скорость ротации
5. **HolyWorld** - для HolyWorld сервера
6. **ReallyWorld** - реалистичная ротация с шумом
7. **Advanced** - продвинутый режим с настройками скорости
8. **Smooth** - очень плавная ротация

### Основные настройки:

#### Дистанция:
- **AttackRange** (1.0 - 6.0) - дистанция атаки
- **LookRange** (0.0 - 3.0) - дополнительная дистанция поиска

#### Режим атаки:
- **ClickMode**: 1.9 / 1.8
- **CPS** (1-20) - клики в секунду для режима 1.8

#### Настройки атаки (AttackSettings):
- ✅ **OnlyCrit** - атаковать только с критами
- ✅ **BreakShield** - умный ломатель щита
- ✅ **NoEatAttack** - не атаковать во время еды
- ✅ **IgnoreWalls** - атаковать через стены
- ✅ **SmartCrits** - умные криты (атака на земле)

#### Дополнительно:
- **HitChance** (1-100%) - шанс удара
- **MovementCorrection**: Free / Focused / None
- **SprintReset**: Legit / Packet

### Настройки для Advanced режима:
- **HorizontalSpeed** (0.1 - 360) - горизонтальная скорость ротации
- **VerticalSpeed** (0.1 - 360) - вертикальная скорость ротации
- **Randomization** (0 - 10) - уровень рандомизации

### Выбор целей (Targets):
- ✅ **Players** - игроки
- ✅ **Mobs** - мобы
- ✅ **Animals** - животные
- ✅ **Friends** - друзья
- ✅ **Invisibles** - невидимые
- ✅ **Villagers** - жители

### Сортировка целей (TargetSort):
- **Distance** - по дистанции
- **Health** - по здоровью
- **Armor** - по броне
- **FOV** - по полю зрения

## 🔧 ТЕХНИЧЕСКИЕ ОСОБЕННОСТИ

### Умный ломатель щита:
- Автоматически находит топор в инвентаре
- Переключается на топор
- Ломает щит
- Возвращается на оружие
- Кулдаун 1.5 секунды

### Система ротации:
- 8 различных режимов наведения
- Плавная интерполяция
- Рандомизация для обхода античитов
- GCD коррекция
- Поддержка движения (Free/Focused/None)

### Система атаки:
- Режим 1.9 (по кулдауну)
- Режим 1.8 (по CPS)
- Проверка критов
- Шанс удара
- Сброс спринта (Legit/Packet)

### Выбор целей:
- Множественные фильтры
- 4 режима сортировки
- Проверка видимости
- Проверка друзей
- Проверка ботов (через AntiBot модуль)

## 🎯 КАК ИСПОЛЬЗОВАТЬ

### Активация модуля:
1. Откройте меню WildClient (обычно Right Shift или Insert)
2. Перейдите в категорию **Combat**
3. Найдите модуль **YanderovKA**
4. Включите модуль

### Настройка:
1. Выберите **AimMode** (рекомендуется FunTime или ReallyWorld)
2. Установите **AttackRange** (обычно 3.0-3.5)
3. Настройте **AttackSettings** под свои нужды
4. Выберите **Targets** (обычно Players + Mobs)
