# ✅ TARGET HUD - ДОБАВЛЕНЫ СТИЛИ!

## 🎨 Что было сделано

Добавлена возможность выбора стиля Target HUD с 3 вариантами на выбор!

---

## 📋 ДОСТУПНЫЕ СТИЛИ

### 1. 🎮 WildClient (Оригинальный)
**Описание**: Оригинальный стиль WildClient с полным функционалом

**Особенности:**
- ✅ Большой аватар игрока (32x32)
- ✅ Отображение брони (4 слота)
- ✅ Полоска здоровья с анимацией
- ✅ HP счетчик
- ✅ Эффект красного свечения при уроне
- ✅ Blur эффект фона
- ✅ Размер: 130x50 пикселей

**Когда использовать:**
- Для полной информации о цели
- Когда нужно видеть броню противника
- Для красивого отображения

---

### 2. 🔥 Yanderov (Стиль из Yanderov)
**Описание**: Адаптированный стиль из Yanderov клиента

**Особенности:**
- ✅ Средний аватар игрока (25x25)
- ✅ Компактная верхняя панель с именем
- ✅ Полоска здоровья
- ✅ HP счетчик
- ✅ WIN/LOSE индикатор (зеленый/красный)
- ✅ Темный фон с blur эффектом
- ✅ Размер: 110x60 пикселей

**Когда использовать:**
- Для стильного отображения
- Когда нужен WIN/LOSE индикатор
- Для среднего размера HUD

---

### 3. 📦 Compact (Компактный)
**Описание**: Минималистичный компактный стиль

**Особенности:**
- ✅ Только имя и здоровье
- ✅ Без аватара
- ✅ Минимальный размер
- ✅ Простая полоска здоровья
- ✅ HP счетчик
- ✅ Темный полупрозрачный фон
- ✅ Размер: 100x35 пикселей

**Когда использовать:**
- Для минимального HUD
- Когда нужно больше места на экране
- Для быстрой информации

## ⚙️ КАК ИСПОЛЬЗОВАТЬ

### В игре:

1. **Откройте настройки HUD**
   - Обычно через меню модулей или специальную кнопку

2. **Найдите Target HUD**
   - В списке HUD элементов

3. **Выберите стиль**
   - Настройка **Style**
   - Варианты: `WildClient`, `Yanderov`, `Compact`

4. **Перетащите HUD**
   - Зажмите и перетащите в нужное место

## 🔧 ТЕХНИЧЕСКИЕ ДЕТАЛИ

### Измененный файл:
```
src/main/java/dev/client/ui/hud/impl/TargetHudElement.java
```

### Добавлено:
- **ModeSetting style** - настройка выбора стиля
- **renderWildClientStyle()** - рендер оригинального стиля
- **renderYanderovStyle()** - рендер стиля Yanderov
- **renderCompactStyle()** - рендер компактного стиля

### Структура кода:
```java
public class TargetHudElement extends HudElement {
    // Настройка стиля
    private final ModeSetting style = new ModeSetting()
            .name("Style")
            .value("WildClient")
            .modes("WildClient", "Yanderov", "Compact");
    
    public void render(DrawContext drawContext) {
        // Выбор стиля
        switch (this.style.getValue()) {
            case "WildClient":
                renderWildClientStyle(drawContext);
                break;
            case "Yanderov":
                renderYanderovStyle(drawContext);
                break;
            case "Compact":
                renderCompactStyle(drawContext);
                break;
        }
    }
}
```

## 💡 РЕКОМЕНДАЦИИ

### Для PvP:
**Рекомендуется**: `WildClient` или `Yanderov`
- Полная информация о противнике
- Видно броню (WildClient)
- WIN/LOSE индикатор (Yanderov)

### Для HvH:
**Рекомендуется**: `Yanderov`
- WIN/LOSE индикатор очень полезен
- Средний размер не мешает
- Стильный вид

### Для минимализма:
**Рекомендуется**: `Compact`
- Минимальный размер
- Только важная информация
- Больше места на экране

### Для красоты:
**Рекомендуется**: `WildClient`
- Самый красивый стиль
- Blur эффекты
- Эффект урона
- Отображение брони
