## 🎮 БЫСТРЫЙ СТАРТ

### Использование Aura:

```
1. Откройте меню модулей
2. Combat → Aura
3. Mode → выберите режим:
   - FunTime (плавный)
   - Matrix (для Matrix)
   - Advanced (настраиваемый)
   - Smooth (очень плавный)
4. ClickMode → 1.8 или 1.9
5. Настройте CPS и HitChance
```

### Использование Speed:

```
1. Откройте меню модулей
2. Movement → Speed
3. Mode → выберите режим:
   - Grim (для Grim античита)
   - Meta (продвинутый)
   - HolyWorld (умный)
   - FunSky HVH (для HvH)
4. Настройте параметры режима
```
