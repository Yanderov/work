## 📋 ЧТО БЫЛО ДОБАВЛЕНО

### ✅ Новые режимы Speed (6 режимов из Yanderov):

1. **Grim** - ускорение от столкновений с сущностями
   - Усиление при наличии цели в Aura
   - Настраиваемая сила столкновений
   - Работает как на Grim античите

2. **HolyWorld** - умное ускорение в зависимости от скорости цели
   - Разное ускорение для медленных и быстрых целей
   - Настраиваемый радиус сканирования
   - Автоматическое определение скорости сущностей

3. **FunTime One Block** - оптимизирован для One Block режима
   - Автоматическая адаптация под эффект Speed
   - Работает только в ограниченном пространстве
   - Фиксированная скорость 0.28-0.32

4. **Meta** - продвинутый режим с множеством настроек
   - 4 уровня скорости в зависимости от эффекта Speed
   - Damage Boost - ускорение при получении урона
   - Collision Boost - ускорение от столкновений
   - Интеграция с Aura для усиления

5. **FunSky HVH** - режим для HvH серверов
   - Адаптивная скорость под эффект Speed
   - Damage Boost система
   - Оптимизирован для FunSky сервера

6. **45Degree** - специальный режим с углом 45°
   - Фиксированная скорость 0.28
   - Работает через пакеты
   - Минимальная нагрузка

## ⚙️ НОВЫЕ НАСТРОЙКИ

### Для режима Grim:
- **AuraBoost** - усиление при наличии цели в Aura (вкл/выкл)
- **Strength** - сила усиления (1.0-6.0)

### Для режима Meta:
- **Speed 0** - скорость без эффекта Speed (0.1-10.0)
- **Speed 1** - скорость с Speed I (0.1-10.0)
- **Speed 2** - скорость с Speed II (0.1-10.0)
- **Speed 3+** - скорость с Speed III+ (0.1-10.0)
- **DamageBoost** - ускорение при уроне (вкл/выкл)
- **BoostDuration** - длительность буста (0-1000 мс)
- **MinDamageSpeed** - минимальная скорость при уроне (0.1-10.0)
- **CollisionBoost** - ускорение от столкновений (вкл/выкл)
- **CollisionPower** - сила столкновений (0.01-0.5)

### Для режима FunSky HVH:
- **DamageBoost** - ускорение при уроне (вкл/выкл)
- **BoostDuration** - длительность буста (0-1000 мс)
- **MinDamageSpeed** - минимальная скорость при уроне (0.1-10.0)

### Для режима HolyWorld:
- **LowBoost** - ускорение для медленных целей (0.0-0.1)
- **HighBoost** - ускорение для быстрых целей (0.0-0.1)
- **ScanRadius** - радиус поиска сущностей (1.0-10.0)

## 🎮 КАК ИСПОЛЬЗОВАТЬ

### В игре:

1. Откройте меню модулей
2. Найдите модуль **Speed** в категории **Movement**
3. Включите модуль
4. Выберите режим в настройке **Mode**

### Выбор режима:

#### Для Grim античита:
```
Mode: Grim
AuraBoost: Включено
Strength: 1.5-2.0
```

#### Для HolyWorld сервера:
```
Mode: HolyWorld
LowBoost: 0.02
HighBoost: 0.036
ScanRadius: 5.0
```

#### Для Meta режима:
```
Mode: Meta
Speed 0: 1.0
Speed 1: 1.5
Speed 2: 2.0
Speed 3+: 2.5
DamageBoost: Включено
CollisionBoost: Включено
```

#### Для FunSky HVH:
```
Mode: FunSky HVH
DamageBoost: Включено
BoostDuration: 200
MinDamageSpeed: 1.5
```

#### Для One Block режима:
```
Mode: FunTime One Block
(Настройки не требуются)
```
